Standard layouts

The standard grid is based off a 12-column grid with 20px gutters between each column

Use a .grid class on a container to set the grid context and .grid-col-n to make columns (where n denotes the number of columns.

Notes:

  • Rows are not needed
  • A column will wrap if it exceeds the number available columns in a row
you get the point

Auto-fill

By using .grid wrapper and .col children, the columns will evenly distribute to fill the whole container.

Notes:

  • Columns will never wrap
  • Caution: You may be able to create layouts that are out of phase with global grid

Offset grid-columns

You can add columns off empty space before a column by adding .offset-n to a .col or .col-n element

n-up layouts

Quickly and easily layout grids with n evenly distributed columns.

Notes:

  • Columns will automatically wrap
  • Columns will always be n/12 of the width of the grid, even when you have fewer

Squashing Gutters

You can squash all gutters for an entire grid by applying the .squash class to the .grid element.

You can squash a single gutter by applying a .squash-next class to any .grid-col/.grid-col-n element.
